Программа поддерживается JVM 1.8 и выше.

Алгоритм работы:
создать массив input файлов
проверить их на существование
определить тип сортировки содержимого файла (по первому неравенству строк, читая с файл с начала)
прочитать строку из каждого файла, добавить в массив
выбрать наименьшее или наибольшее значение массива (в зависимости от типа сортировки) и записать его в output файл
проверить значение следующей (nextLine) строки input файла, добавить его в массив строк (currentLine)
повторять последние 3 пункта до тех пор, пока массив строк (currentLine) не станет пустым

В случае, если файл сортирован по возрастанию, а выполнить необходимо сортировку по убыванию, то чтение файла
будет осуществляться с конца.
Программа может обрабатывать файлы с некорректной информацией, чтение данных будет осуществляться до момента ошибки.

Логирование ведется в файл log.out в корневой каталог проекта.
